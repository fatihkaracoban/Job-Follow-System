Bu proje, işletmelerin veya bireysel çalışanların yürüttükleri **iş süreçlerini** ve bu işler için kullanılan **malzeme giderlerini** dijital ortamda kolayca takip etmeleri için geliştirilmiş bir web uygulamasıdır.

## ✨ Öne Çıkan Özellikler

* **📊 İş Yönetimi:** Yeni iş tanımlama, durum güncelleme (Yapılacak, Devam Ediyor, Tamamlandı) ve ücret takibi.
* **📦 Malzeme Entegrasyonu:** Her işe özel malzeme listesi oluşturma, adet ve birim fiyat üzerinden otomatik maliyet hesaplama.
* **✏️ Tam CRUD Desteği:** İşler ve malzemeler üzerinde tam kontrol (Ekleme, Listeleme, Güncelleme, Silme).
* **🏠 Modern Arayüz:** Kullanıcı dostu ikonlar (Font-Awesome) ve hızlı navigasyon sağlayan ana sayfa erişimi.
* **⚡ Dinamik Hesaplamalar:** Malzeme birim fiyatı ve adedi değiştikçe otomatik güncellenen toplam maliyet tablosu.

## 🛠️ Kullanılan Teknolojiler

* **Backend:** Java 17+, Spring Boot 3.x, Spring Data JPA
* **Frontend:** Thymeleaf, HTML5, CSS3, Font-Awesome İkonları
* **Veritabanı:** MySQL / H2 Database
* **Araçlar:** Maven, Lombok

## 🚀 Kurulum

1.  Projeyi klonlayın:
    ```bash
    git clone [https://github.com/fatihkaracoban/Job-Follow-System.git](https://github.com/fatihkaracoban/Job-Follow-System.git)
    ```
2.  `application.properties` dosyasındaki veritabanı ayarlarını kendi sisteminize göre güncelleyin.
3.  Projeyi Maven ile derleyin:
    ```bash
    mvn clean install
    ```
4.  Uygulamayı çalıştırın ve tarayıcıdan `http://localhost:8080` adresine gidin.
